Como o mercado digital se atualiza de modo acelerado, atualmente existem vários tipos e modelos de páginas web, com fins diferenciados, com o objetivo de atender a necessidade de cada cliente em particular. Confira alguns dos modelos mais utilizados.

Site estático

Sites estáticos normalmente contém um conjunto fixo de páginas com um layout específico. Ao executar uma página estática em um navegador, o conteúdo não muda em resposta conforme as ações do usuário. Um site estático é desenvolvido com uma linguagem de marcação HTML e CSS, podemos escrever esses códigos utilizando editores gratuitos como por exemplo, o Visual Studio Code da Microsoft, e outros editores disponíveis na internet.

Site one page

One page ou (página única) em português, é um tipo de página web que possui um único layout, podemos encontrar todas as informações como: "sobre", "serviços", "contatos" e muito mais.

Landing page

Landing page, ou também conhecida como página de aterrisagem, captura ou conversão, é bastante útil na estratégia de markting digital, otimizada para gerar leads, etc.

HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XTML.

JavaScript

JavaScript (frequentemente abreviado como JS) é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma (protótipos, orientado a objeto, imperativo e funcional).
